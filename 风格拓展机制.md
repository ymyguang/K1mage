整体需求可以这样概括：

后端需要支持一套可扩展的 Prompt 模板管理机制。每个 Prompt 模板作为一个独立能力/商品存在，包含模板内容、效果图、价格、别名、排序信息等配置。后续新增、下架、调价、调整展示顺序、修改别名时，尽量不改后端代码，只通过配置文件完成。

⸻

具体方案

1. 文件夹结构

prompts/
├── style_001/
│   ├── prompt.txt      # Prompt 模板内容
│   ├── preview.png     # 效果图
│   ├── price.json      # 价格配置
│   └── meta.json       # 名称、别名、排序、状态等信息
│
├── style_002/
│   ├── prompt.txt
│   ├── preview.png
│   ├── price.json
│   └── meta.json

⸻

2. meta.json 示例

{
  "id": "style_001",
  "name": "工业结构风",
  "alias": ["工业风", "机械风", "结构设计"],
  "description": "强调结构、线条和冷色调的工业视觉风格",
  "order": 100,
  "click_count": 0,
  "is_active": true,
  "is_featured": false
}

说明：

id：模板唯一标识，对应文件夹名
name：前端展示名称
alias：别名/搜索关键词
description：模板说明
order：人工排序，数值越大越靠前
click_count：点击量，可用于热门排序
is_active：是否启用展示
is_featured：是否推荐

⸻

3. price.json 示例

{
  "price_per_image": 5,
  "currency": "CNY"
}

⸻

4. prompt.txt 示例

你是一位专业的视觉设计师，擅长工业结构风格（Industrial Style）的图像设计。
请根据用户的输入内容，生成一段高质量的图像生成提示词，用于AI图像生成模型。
【风格要求】
- 强调工业设计感、机械结构、理性秩序
- 线条清晰、结构分明，具有工程感
- 色调以冷色为主，低饱和度
- 光影硬朗，突出体积与结构
【输出要求】
- 输出为一段英文prompt
- 不要解释，不要加说明
- 不要出现中文
【用户输入】
{input}

⸻

5. 后端加载逻辑

后端启动时扫描 prompts/ 目录：

读取每个 style_xxx 文件夹
↓
加载 prompt.txt
↓
加载 preview.png 路径
↓
加载 price.json
↓
加载 meta.json
↓
过滤 is_active = true 的模板
↓
按 order / click_count 排序返回给前端

⸻

6. 排序规则

推荐支持两种：

默认排序：order 越大越靠前
热门排序：click_count 越大越靠前

后续也可以做混合排序：

综合分 = 人工排序权重 + 点击量权重

⸻

7. 这个方案的好处

新增模板：新增一个 style_xxx 文件夹即可
修改价格：改 price.json
调整排序：改 meta.json 里的 order
修改别名：改 alias
下架模板：is_active 改为 false
替换效果图：替换 preview.png

核心价值是：

Prompt 模板不写死在代码里，而是配置化管理，方便后续扩展、运营和维护。
